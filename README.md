# 💻Programmer
>👨🏻‍💻 사용자가 문제를 선택하여 Java 언어로 직접 코드를 작성, 제출 할 수 있는 플랫폼

<br><br>

## 선정 이유 
>해당 프로젝트는 알고리즘 문제 풀이 기반의 학습 플랫폼을 직접 설계하고 구현함으로써, 백엔드 전반에 대한 실무 역량을 향상시키기 위해 선정했습니다.
>문제 등록, 검색, 코드 제출, 채점 결과 처리 등 복잡한 기능들을 직접 구현하며 RESTful API 설계, JPA 기반의 데이터 처리, 세션/인증 관리 등 실무에서 자주 사용되는 핵심 기술을 경험할 수 있었습니다.
>또한, 사용자 경험을 고려한 UI/UX와 페이징, 검색, AJAX 통신 등 프론트엔드와 백엔드의 통합 개발 경험을 쌓을 수 있는 좋은 기회였습니다.
>이 프로젝트를 통해 단순한 CRUD를 넘어, 도메인 중심 설계(Domain-Driven Design) 및 문제 풀이 상태 관리, 사용자 통계 처리 로직 등 실제 서비스 수준의 구조 설계와 성능 고려까지 고민할 수 있었습니다.

<br><br>

## 주요 기능
### 문제 풀이 목록 및 검색 기능
- 제목 기반 검색 ( AJAX ) 
- 페이징 처리된 문제 리스트
- 난이도, 해결한 사람 수, 문제 풀이 상태 표시 

### 문제 풀이 페이지
- 문제 설명, 입출력 예시, 제한 사항 등 제공
- 코드 작성 및 제출 

### 회원 기능
- 유저 별 푼 문제 확인 
- 유저 순위, 점수, 컴파일 횟수 표시

<br><br>

## 기술 스택
- Frontend : HTML, CSS, JavaScript
- Template : Thymeleaf
- Backend : Spring Boot, Spring MVC, JPA
- Database : Oracle
- Build Tool : Gradle
