# 💻Programmer
>👨🏻‍💻 사용자가 문제를 선택하여 Java 언어로 직접 코드를 작성, 제출 할 수 있는 플랫폼

**Programmers**는사용자가 문제를 선택하여 Java 언어로 직접 코드를 작성, 제출 할 수 있는 플랫폼입니다.

<br><br>

##주요 기능
###문제 풀이 목록 및 검색 기능
- 제목 기반 검색 ( AJAX ) 
- 페이징 처리된 문제 리스트
- 난이도, 해결한 사람 수, 문제 풀이 상태 표시 

###문제 풀이 페이지
- 문제 설명, 입출력 예시, 제한 사항 등 제공
- 코드 작성 및 제출 

###회원 기능
- 유저 별 푼 문제 확인 
- 유저 순위, 점수, 컴파일 횟수 표시

<br><br>

##기술 스택
Frontend : HTML, CSS, JavaScript
Template : Thymeleaf
Backend : Spring Boot, Spring MVC, JPA
Database : Oracle
Build Tool : Gradle
